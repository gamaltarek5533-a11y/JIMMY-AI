API

ادمج الذكاء الاصطناعي في دقائق

API بسيط وقوي يتيح لك بناء تطبيقات AI متطورة بكود أقل وكفاءة أعلى. يدعم Python، JavaScript، وأكثر.

احصل على API Key الوثائق
example.js JAVASCRIPT 
// تكامل JIMMY AI في ثوانٍ
import jimmy from '@jimmy/sdk';

const client = new jimmy({
  apiKey: process.env.JIMMY_KEY
});

const res = await client
  .chat.complete({
    model:   'jimmy-pro-v2',
    message: 'حلّل هذا المشروع',
    lang:    'ar'
  });
